    Listas (project v0.1.0)

    EJEMPLOS

    Ejemplo 1: Declaración de una lista

    iex(1)> list = [3, 4, :atomo, "Hola"]
[3, 4, :atomo, "Hola"]

    Ejemplo 2: Añadir elementos al principio de la lista

    iex(2)> ["Nuevo elemento" | list]
["Nuevo elemento", 3, 4, :atomo, "Hola"]

    Ejemplo 3: Concatenacion de listas

    iex(3)> list = list ++ ["Elem1", "Elem2"]
[3, 4, :atomo, "Hola", "Elem1", "Elem2"]
iex(4)> list = list ++ ["Elemento final"]
[3, 4, :atomo, "Hola", "Elem1", "Elem2", "Elemento final"]

    Ejemplo 4: Sustraccion de listas (utilizando comparación estricta)

    iex(5)> lista = [ 3, 4, :atomo, "Hola"] -- [3,4]
[:atomo, "Hola"]
iex(6)> lista = [ 3, 4.154] -- [3, 4]
[4.154]

    Ejemplo 5: Funciones Head y Tail

    iex(7)> lista = hd [ 3, 4, :atomo, "Hola"]
3
iex(8)> lista = tl [ 3, 4, :atomo, "Hola"]
[4, :atomo, "Hola"]
