    Operador_Pipe (project v0.1.0)

    Ejemplos

    Dado el siguiente código:

    iex> string = "hello, world!"
"hello, world!"
iex> words = String.split(string, " ")
["hello,", "world!"]
iex> capitalized_words = Enum.map(words, &String.capitalize/1)
["Hello,", "World!"]
iex> Enum.join(capitalized_words, " ")
"Hello, World!"

    Vamos a implementar el operador pipe para obtener un código más limpio:

    iex> "hello, world!" |> String.split(" ") |> Enum.map(&String.capitalize/1) |> Enum.join
"Hello, World!"

    Vamos a crear un módulo llamado 'Asignar Error' para manejar errores en el siguiente código.

    defmodule AsignarError do
  def ass("") do
    {:error,"No se ha asignado valor"}
  end
  def ass(v) do
    {:ok,"El valor asignado es ->",v}
  end
end

iex(1)> "" |> AsignarError.ass()
{:error, "No se ha asignado valor"}

iex(2)> string = case (27 |> AsignarError.ass()) do
...(2)>   {:ok, "El valor asignado es ->", 27} -> "El valor asignado es -> 27"
...(2)>   {:error,"No se ha asignado valor"} -> "es el valor asociado a"
...(2)> end
iex(3)> string |> String.upcase() |> String.split()
["EL", "VALOR", "ASIGNADO", "ES", "->", "27"]
