    comparador_tipos(v1, v2)

    Specs

    comparador_tipos(any(), any()) :: String.t()

    Comparación de variables de distintos tipos.

    Parámetros

    • v1: Variable a comparar número 1 (cualquier tipo)
    • v2: Variable a comparar número 2 (cualquier tipo)

    Examples 

    iex(1)> numero = 1

    iex(2)> atomo = :boo

    iex(3)> Introduccion.comparador_tipos(numero, atomo)
"La variable v1: 1 es menor que v2: :boo"

iex(4)> tupla = {1,2,3}
iex(5)> Introduccion.comparador_tipos(atomo, tupla)
"La variable v1: :boo es menor que v2: {1, 2, 3}"

    son_anagramas(s1, s2)

    Función: Son anagramas. Dadas dos cadenas devuelve true si son anagramas, false en otro caso.

    Parámetros

    • s1: String
    • s2: String

    Examples 

    iex(1)> Introduccion.son_anagramas("ana", "naa")
true
iex(2)> Introduccion.son_anagramas("ana", "nAa")
true
iex(3)> Introduccion.son_anagramas("ana", "nva")
false

    sort_string(st)

    Función sort: Ordena una cadena, sustituyendo las letras mayúsculas por minúsculas. Output: Lista de caracteres ordenadas.

    Parámetros

    • st: String

    Examples 

    iex(1)> Introduccion.sort_string("ignacio")
["a", "c", "g", "i", "i", "n", "o"]
iex(2)> Introduccion.sort_string("IgnaciO")
["a", "c", "g", "i", "i", "n", "o"]
