    EJEMPLOS

    Ejemplo 1: Evaluar una funcion de manera asíncrona

    iex(1)> defmodule ModuloEjemplo do
...(1)>   def mult(op1,op2) do
...(1)>     IO.puts(op1*op2)
...(1)>   end
...(1)> end
iex(2)> spawn(ModuloEjemplo, :mult, [2, 3])
6
#PID<0.117.0>

    Ejemplo 2: Enlace de procesos

    iex(3)> defmodule EjemploEnlaceProcesos do
...(3)>   def bye, do: exit("ESTA ES LA RAZON POR LA QUE HEMOS SALIDO")
...(3)>   def run do
...(3)>     Process.flag(:trap_exit, true)
...(3)>     spawn_link(EjemploEnlaceProcesos, :bye, [])
...(3)>     receive do
...(3)>       {:EXIT, from_pid, reason} -> IO.puts("spawn_link: terminado por la razon: " <> reason)
...(3)>     end
...(3)>   end
...(3)> end
iex(4)> EjemploEnlaceProcesos.run
spawn_link: terminado por la razon: ESTA ES LA RAZON POR LA QUE HEMOS SALIDO

    Ejemplo 3: Spawn_monitor para comunica procesos sin utilizar enlace (link)

    iex(5)> defmodule EjemploSinEnlaceProcesos do
...(5)>   def bye, do: exit("ESTA ES LA RAZON POR LA QUE HEMOS SALIDO")
...(5)>   def run2 do
...(5)>     {pid, ref} = spawn_monitor(EjemploEnlaceProcesos, :bye, [])
...(5)>     receive do
...(5)>       {:DOWN, ref, :process, from_pid, reason} -> IO.puts("spawn_monitor: terminado por la razon: " <> reason)
...(5)>     end
...(5)>   end
...(5)> end
iex(6)> EjemploSinEnlaceProcesos.run2
spawn_monitor: terminado por la razon: ESTA ES LA RAZON POR LA QUE HEMOS SALIDO
:ok

    Ejemplo 4: Agentes

    iex(7)> tuple = {"val1", 2, "val3"}
iex(8)> {:ok, agent} = Agent.start_link(fn -> (tuple) end)
iex(9)> Agent.update(agent,fn({a,b,c})->{c,a,b} end)
:ok
iex(10)> Agent.get(agent, &(&1))
{"val3", "val1", 2}
