    Enum_ (project v0.1.0)

    EJEMPLOS

    Ejemplo 1: Enum.all

    iex(1)> bool = Enum.all?([50, 20, 10], fn(s) -> s >= 15 end)
false
iex(2)> bool = Enum.all?([1==1, true, "x"=="x"], fn(s) -> s >= 15 end)
true

    Ejemplo 2: Enum.any

    iex(3)> bool = Enum.any?([50, 20, 10], fn(s) -> s >= 15 end)
true
iex(4)> bool = Enum.any?([true, 3, 10], fn(s) -> s >= 15 end)
true

    Ejemplo 3: Chunk_by

    iex(5)> bool = Enum.chunk_by(["uno", "dos", "tres", "cuatro", "cinco", "cien"],fn(x) -> String.length(x) end)
[["uno", "dos"], ["tres"], ["cuatro"], ["cinco"], ["cien"]]

    Ejemplo 4: Map

    iex(6)> list = Enum.map([0, 1, 2, 3], fn(x) -> x - 1 end)
[-1, 0, 1, 2]

    Ejemplo 5: Filter

    iex(7)> list = Enum.filter([1, 2, 3, 4], fn(x) -> rem(x, 2) == 1 end)
[1, 3]

    Ejemplo 6: Min

    iex(8)> lista = Enum.min([1,2,3,4,1])
[4, :atomo, "Hola"]

    Ejemplo 7: Max

    iex(9)> lista = Enum.max([9,2,3,4,2])
9

    Ejemplo 8: Reduce

    iex(10)> lista = Enum.reduce([5,3,1],-7, fn(numero,contador) -> numero*contador end)
-105

    Ejemplo 9: Sort

    iex(11)> lista = Enum.sort([:ok,:okkkkkk,2,3,5])
[2, 3, 5, :ok, :okkkkkk]
