    Recursividad (project v0.1.0)

    Ejemplos

    Ejemplo 1: bucle simple para multiplicar dos operandos

    iex(1)> defmodule EjemploBucleSimple do
...(1)>   def bucle(almacen, x, y) when y <= 1 do
...(1)>     IO.puts almacen + x
...(1)>    end
...(1)>    def bucle(almacen, x, y) do
...(1)>      bucle(almacen + x, x, y-1)
...(1)>     end
...(1)> end
iex(2)> EjemploBucleSimple.bucle(0,6,5)
30
:ok

    Ejemplo 2: Suma de los elementos de una lista

    iex(2)> defmodule EjemploSumaLista do
...(2)>   def buc([x|xs], almacen) do
...(2)>     buc(xs, x + almacen)
...(2)>   end
...(2)>   def buc([], almacen) do
...(2)>     almacen
...(2)>   end
...(2)> end
iex(3)> EjemploSumaLista.buc([1,2,3,4],0)
10
